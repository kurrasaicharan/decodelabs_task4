## 📌 About The Project

**General Knowledge Quiz** is a beginner-friendly Python command-line application designed to test users' general knowledge skills through an interactive quiz system.

The application presents multiple questions, validates user answers, calculates the final score, and provides feedback based on the user's performance.

This project focuses on implementing fundamental Python concepts such as variables, conditional statements, loops, user input handling, and basic program logic.

## 🎯 Project Objectives

- Build an interactive quiz application using Python
- Practice decision-making using conditional statements
